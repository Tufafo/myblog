## 时间属性

| 语法      | 含义                        |
| :------ | ------------------------- |
| `[]P`   | P总为真                      |
| `<>P`   | P最终为真                     |
| `[]~P`  | P总为假                      |
| `~[]P`  | P不总为真，有假的情况               |
| `<>[]P` | 最终P总为真（中间状态可以为假，但首尾状态应为真） |
| `[]<>P` | P总最终为真                    |
